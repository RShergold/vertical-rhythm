Layout Rules

3 - first rule

So we know we want to keep our margins from touching. Lets start with the first rule

block elements to only have a bottom margin - never top

By making sure we never use top margin on our layout elements we ensure we know with certainty where the spacing is coming from

    .div1, .div2 {
      height: 18px;
    }
    .div1 {
      background-color: blue;
      margin-bottom: 18px;
    }
    .div2 {
      background-color: green;
      margin-bottom: 36px;
    }

4 - Setting font sizes

Font Size

We are going to set the font size ( new-font-size ) of our H1 tags to 24px.
All of our font sizes are set in em's - relative to our base font size of 12px - we do a simple calculation.

new-font-size ÷ base-font-size = h1-font-size
24 ÷ 12 = 2

so we set our font-size to 2em. 

    h1 {
      font-size: 2em
    }

Line Height

Next up we set our line height. We want this to fit our guide lines so needs to be a multiple of our base-line-height.
first we calculate how many lines our new font size will need.

ceil( new-font-size ÷ base-line-height ) = lines-needed
ceil( 24 ÷ 18 ) = 2

we now know we need 2 lines to fit a font of 24px

Next we calculate the actual line-height of our H1 tag.
This is in em's in relation to the new font-size (24px) and wants to equal 2 lines.

( base-line-height × lines-needed ) ÷ new-font-size = h1-line-height
( 18 × 2 ) ÷ 24 = 1.5

so we now set out H1 line-height to 1.5em 

    h1 {
      font-size: 2em;
      line-height: 1.5em;
    }

Now do that in Compass

Luckily in compasss we have a simple mixin to call that will do all that for us.
Simply include adjust-font-size-to and set the size in pixels.

style.scss 
    h1 {
      @include adjust-font-size-to(24px);
    }

    h2 {
      @include adjust-font-size-to(18px);
    }

    h3 {
      @include adjust-font-size-to(13px);
    }

compiles to:

style.css 
    h1 {
      font-size: 2em;
      line-height: 1.5em;
    }

    h2 {
      font-size: 1.5em;
      line-height: 2em;
    }

    h3 {
      font-size: 1.08333em;
      line-height: 1.38462em;
    }

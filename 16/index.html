6 - the third rule

We now know that, where the bottom margins meet and the margin collapses, the solution is to change the block formatting context

we can do this with a standard css trick, clearfix

Lets create a nested div structure (note: our P elements already have a bottom-margin of 18px )

    .frame {
      padding-top: 18px;
      background-color: rgba(0,128,128, 0.2);
      margin-bottom: 36px;
    }

    <div class="frame">
      <p>content here</p>
      <div class="frame">
        <p>content here</p>
        <div class="frame">
          <p>content here</p>
        </div>
      </div>
    </div>

Part II

Layout Rules

The following technique was developed by Mark Kirby, it serves a number of useful purposes.

  • Ensures we don't run into any problems with collapsing margins
  • Avoids a couple of hasLayout issues
  • gives all of our widgets a uniform structure meaning we can swap
    them in and out or around the page without breaking the flow.
  • It means a number of developers can work on the same page with an
    immediate understanding of how the page is structured.

Vertical Rhythm

7 - borders

When you add a border to an element it will of corse affect it's height

style.scss 
          .left {
            color: blue;
            border-bottom: 4px solid;
          }
Test line for .left class

As you can see the border is added to the bottom of our DIV element, but this adds 4px to the elements height and throws off the rhythm for all of the following elements.

We can use the mixin trailing-border(). This will add padding of one row (minus the border height) to the bottom of our element and place the new border at the bottom of that row.

style.scss 
          .right {
            color: green;
            @include trailing-border(4px)
          }

Compiles to

style.css 
          .right {
            color: green;
            border-bottom-style: solid;
            border-bottom-width: 0.33333em;
            padding-bottom: 1.16667em;
          }
Test line for .right class

we can now continue below our border in rhythm

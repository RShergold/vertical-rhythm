Layout Rules

4 - second rule

when we nest elements the only top spacing should be in the form of padding in the container

    .div1, .div2 {
      height: 18px;
    }
    .div1 {
      background-color: blue;
      margin-bottom: 18px;
    }
    .div2 {
      background-color: green;
      margin-bottom: 36px;
    }
    .div3 {
      padding-top: 18px;
      background-color: rgba(1, 1, 1, 0.2);
      margin-bottom: 36px;
    }

Now we can see that we (correctly) have a space at the top of our frame (div3) and that each inner div has its correct under spacing

However there is a major problem at the bottom of the container.
the bottom margin of of the container (div3) meets the bottom margin of the last row (div1).
because of this the smaller margin (div1) is collapsed.

with our layout rules this will be the only place where our margins can collapse.

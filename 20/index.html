Putting it all together

2 - add some spacing classes

Next up we are going to add some classes for adding vertical spaces to elements.


The u- prefex stands for Utility. This explains the class' category and where to find it in our normal css file structure
(for readability; we have added everything to one file here)


SCSS compiles to CSS 
    //u-rhythm
    .u-rhythmTop {
      padding-top: rhythm(0.5);
    }
    .u-rhythmTop-1 {
      padding-top: rhythm(1);
    }
    .u-rhythmTop-2 {
      padding-top: rhythm(2);
    }
    .u-rhythmTop-3 {
      padding-top: rhythm(3);
    }

    .u-rhythmBottom {
      @include trailer(0.5);
    }
    .u-rhythmBottom-1 {
      @include trailer(1);
    }
    .u-rhythmBottom-2 {
      @include trailer(2);
    }
    .u-rhythmBottom-3 {
      @include trailer(3);
    }

We can now space elements by simply adding u-rhythm* classes


u-rhythmBottom-1
u-rhythmBottom -- we can also space things in fractions of a rhythm

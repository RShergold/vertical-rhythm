Setting up our environment

We will be using the Vertical Rhythm tools in Compass. To use this we will need SASS.

Your setup may be different but i'm using a Rake file to start sass watching for changes in the repo and compiling .scss files to .css
My rake file looks like this.

Rakefile 
    desc 'Watch SCSS'
    task :watch_scss do
        current_location = File.dirname(__FILE__)
        sh "compass watch --sass-dir #{current_location} --css-dir #{current_location} -I #{current_location} -e production"
    end

In the terminal I can now navigate to the repository root file and run rake watch_scss.
Any time I create/change a .scss file SASS will now compile the sass into a css file of the same name in the same directory.

2 - collapsing margins continued

now consider nesting margins. We use the same css but nest the divs this time 

    .div1, .div2 {
      height: 18px;
    }
    .div1 {
      background-color: blue;
      margin: 18px;
    }
    .div2 {
      background-color: green;
      margin: 36px;
    }

    <div>div ------------------------ no padding or margin</div>
    <div class="div1">
      <div class="div2"></div>
    </div>
    <div>div ------------------------ no padding or margin</div>
div ------------------------ no padding or margin
div ------------------------ no padding or margin


So what happened here??
The first thing to note is that only top and bottom margins are collapsed, so the side margins act as expected.
The top margins are touching so the smaller margin ( div1 ) is collapsed and we are only using the larger margin ( div2 )
The bottom margin is small because we are fixing the height of div1. If we remove that height, we can see the bottom margin
also takes on the height of div2. 

    <div>div ------------------------ no padding or margin</div>
    <div class="div1" style="height:initial">
      <div class="div2"></div>
    </div>
    <div>div ------------------------ no padding or margin</div>
div ------------------------ no padding or margin
div ------------------------ no padding or margin


however, this still does not give us what we may of been expecting.

One of the standard hacks to fix this is to ensure the margins never meet eachother. This can be done by adding a small amount of padding to the element. 

    <div>div ------------------------ no padding or margin</div>
    <div class="div1" style="height:initial;padding:1px">
      <div class="div2"></div>
    </div>
    <div>div ------------------------ no padding or margin</div>
div ------------------------ no padding or margin
div ------------------------ no padding or margin


the problem here is we have now thrown our elements out of rhythm.
However we have picked up an important concept. Ensuring vertical margins never meet means we can be sure we never encounter collapsing margins.

So we stick to two simple rules to help us out.

Vertical Rhythm

1 - Setting up Vertical Rhythm

the first thing we do is import vertical rhythm from compass

As most browsers default to a font-size of 16px we are going to change ours to something noticeably different.
We are going with a font-size of 12px and line-height of 18px.
finally, to actually apply these values we call the establish-baseline mixin

style.scss 

    @import "compass/typography/vertical_rhythm";

    $base-font-size: 12px;
    $base-line-height: 18px;

    @include establish-baseline;

Compiled scss

Our style.scss file compiles into the following

style.css 
    * html {
      font-size: 75%;
    }

    html {
      font-size: 12px;
      line-height: 1.5em;
    }

The first 3 lines are aimed at Internet Explorer 6 and older.
Here we set the base font size as a percentage of the default (16px).
75% of 16 = 12

Our next line sets the base font size in pixels to our chosen size. (12px)

Finally the base line height is set.
This is done in em's which is relative to the font size.
1em is now 12px (our font size) so to set our line height to 18px we set line height to 1.5em
12 * 1.5 = 18
